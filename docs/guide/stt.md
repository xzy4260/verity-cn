# STT 语音识别

按住说话，Verity 将听到你的声音。

## 模式对比

| | 本地识别 (LOCAL) | MiMo API (MIMO) |
|---|---|---|
| 引擎 | Whisper tiny.en (Sherpa-ONNX) | MiMo ASR API |
| 语言 | 仅英文 | 中英文 |
| 网络 | 离线 | 需要联网 |
| 平台 | 仅桌面 | 桌面 + 安卓 |

## 桌面端

本地模式首次启动自动解压 Whisper tiny.en 模型（约 50MB）。支持英文语音识别。

如需中文识别，请使用 MiMo ASR。

## 安卓端

**仅支持 MiMo API 模式**。安卓通过 LWJGL 的 ALC Capture 扩展采集麦克风，实时显示音量电平条。

> 需要启动器授予**麦克风权限**。在系统设置 → 应用 → 启动器 → 权限中开启。

## 按键说明

长按 `V` 键说话是游戏默认设置。如果按住没反应：
- 检查按键设置 → 「语音输入」是否与其他模组冲突
- 可在「选项 → 按键设置」中自定义为其他键位

## 电平条

按住说话时屏幕中央显示音量电平条，随声音大小实时变化。绿色表示正常，红色表示过载。始终不动 = 麦克风未工作。
