## 关于

**作者：** xzy4260 · [B站频道](https://space.bilibili.com/3706993095215271)

**v3.0 合作开发：** [B站 涓星向凡](https://space.bilibili.com/3461565078571133)

Verity 是一个 Minecraft Forge 1.20.1 模组，添加 AI 驱动的伴侣角色。本版基于原版 5.7.2 进行中国化改造：

- 删除硬编码的 Groq/OpenRouter 配置
- 新增 LLM / TTS / STT 三板块配置系统
- 全中文 GUI + 语言包
- 安卓平台完整适配
- MiMo v2.5 系列深度集成
