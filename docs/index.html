更新日志

v2.75 — 模型自定义 + 皮肤系统

2026-07-21
  • TTS/STT 模型名可配置：用户手动输入，默认 mimo-v2.5-tts / mimo-v2.5-asr，兼容所有 OpenAI 模型
  • 自定义皮肤系统：将 16 张贴图打包为 ZIP 放入 config/verity/skins/，GUI 中输入 ZIP 名即时切换
  • 皮肤修复：修复表情固定不变、物品栏不显示自定义皮肤的 bug

v2.5 — 中国化重构更新

2026-07-21
  • AI 配置完全重构：删除旧的 Groq/OpenRouter 硬编码配置，新增 LLM / TTS / STT 三板块独立配置
  • LLM 板块：支持任意 OpenAI 兼容 API（DeepSeek、通义千问、硅基流动等）
  • TTS 板块：三种模式 — 内置 Piper 引擎 / Base URL 自定义 API / MiMo 音色克隆
  • STT 板块：两种模式 — 本地 Whisper 识别 / Base URL 自定义 API
  • 小米 MiMo 深度集成：支持 mimo-v2.5-tts（预置音色）、mimo-v2.5-tts-voiceclone（音色克隆）、mimo-v2.5-asr（语音识别），填入 xiaomimimo.com 地址自动兼容
  • 全中文配置界面：OptionGroup 分组、选项名、描述全部本地化
  • 配置界面重构：新增 VerityClientBootstrap 引导类 + VerityClient 拆至 gui 子包，解决 Gradle 编译顺序问题
  • 枚举显示修复：TtsMode / SttMode / CloneVoice / MimoApiEndpoint 全覆盖 toString()，确保下拉菜单显示中文

v2.0 — 首版中国化

2026-07-17
  • 完整 zh_cn.json 中文语言文件
  • AI 配置界面初步汉化
