# 2026-07-19

## 安卓适配全部完成

### TTS
- 桌面保持 javax.sound.SourceDataLine
- 安卓使用 LWJGL OpenAL (alGenBuffers/alBufferData/alSourcePlay)
- 关键修复: `AL_SOURCE_RELATIVE=true` + `AL_GAIN=1.0f` 解决 OpenSL 后端无声问题

### STT 麦克风
- 方案演进: AudioRecord 反射(ClassNotFoundException) → JNA(NoClassDefFoundError, Android ARM64 缺 libjnidispatch) → LWJGL JNI 直调 ALC capture 函数指针
- 最终方案: ZalithMicBridge 通过反射匹配 JNI.invokePJ/invokePV/invokePPV/invokePI 调用 ALCCapabilities 里的函数指针
- 采用流式录音循环(androidRecordLoop)替代一次性阻塞录音，实现实时电平条
- PCM→WAV 改用手动写头(pcmToWav)，不依赖 javax.sound.AudioSystem

### 平台检测
- ZalithLauncher + PojavLauncher + FCL 全系兼容
- 通过 user.home / java.library.path 路径 + 类名反射检测

### 构建坑
- Gradle clean 被 safe-delete 阻止，导致多次输出旧版 JAR
- 解决方案: `gradlew clean` 命令清理
- Python zipfile mode='a' 追加造成重复条目，classloader 读第一个(旧版)
- 解决方案: mode='w' 从零构建，先复制 verity-patched.jar 条目再覆盖新类
